This is a description of the files in this directory.

ml_new_*.csv	-	Data set used to train/test the algorithm.
preProcess.py	-	Python2. Holds preprocessing and processing functions used for the Raiccho method.
tfidf.py	-	Python2. Executes the Raiccho method. Run this file by using, "execfile('tfidf.py"). No inputs are necessary. Returns a CCR on the specified dataset. Inputs/parameters can be modified from within the file.
